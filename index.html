Machine Learning Roadmap

Learning layer, later

ATLAS should become machine-learning aware only after the source model is reliable. The future learning layer should learn from structured documents, user notes, concept links, and verified citations.

Stage 1: Structured memoryStore articles, paragraphs, annotations, bookmarks, concepts, and source anchors.
Stage 2: Retrieval intelligenceUse semantic search and embeddings to find relevant source units without inventing facts.
Stage 3: Personal study modelAdapt revision prompts to what the student reads, saves, misunderstands, and revisits.
